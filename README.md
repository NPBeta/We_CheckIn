#We 重邮 自动打卡脚本
------
基于 Python 3 的 We 重邮自动打卡脚本
------

我是懒狗，代码想到哪写到哪，若可读性较差请见谅
目前仅根据返回的 JSON 判断是否打卡成功，可能有误判

**使用方法**
* 安装 requests 模块
* 在脚本里写死姓名和学号（可选）
* 运行 run.py
* 脚本会提示是否打卡成功
